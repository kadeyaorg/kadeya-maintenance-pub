# kadeya-maintenance-pub

**Public** repository for **kadeya-maintenance** distribution (desktop installers, the auto-update manifest, and HTML documentation).

Source code and development live in the private `kadeya-maintenance` repository within the organization; this repo does not contain application sources, only what end users and machines need. The app is a cross-platform Tauri + React build (Windows today; macOS/Linux ready).

## Releases

Signed desktop installers and the updater manifest (`latest.json`) are published here by the
release workflow in the private source repo:

**[Releases — kadeyaorg/kadeya-maintenance-pub](https://github.com/kadeyaorg/kadeya-maintenance-pub/releases)**

To migrate from the legacy Go app, install the latest v2 installer once; the desktop app then
auto-updates from this repo.

## Documentation (GitHub Pages)

Interface documentation (HTML) is in the [`docs/`](./docs/) folder. Table of contents: [`docs/index.html`](./docs/index.html).

To publish the site:

1. **Settings → Pages**
2. **Build and deployment**: *Deploy from a branch*
3. Branch `main`, folder **`/docs`**
4. Site URL: `https://kadeyaorg.github.io/kadeya-maintenance-pub/`

The HTML docs in `docs/` are maintained by hand in this repo (they are not generated by the
release workflow).
