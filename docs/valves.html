Checkboxes

Select one or more valves to open. Names map to outputs: Main, Still, Carb, Flavor 1–4.

Buttons

Open selected
Builds JSON with each valve state: true for checked (open), false for others, then publishes to hmi/actuators_request.
Close all
Sends the command that closes all valves (regardless of checkboxes).

Exact JSON details are defined in the application module; if unsure, correlate with the MQTT monitor.
