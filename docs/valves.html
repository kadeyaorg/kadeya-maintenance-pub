Cases à cocher

Sélectionnez une ou plusieurs vannes concernées par l’ouverture. Les noms correspondent aux sorties : Main, Still, Carb, Flavor 1 à 4.

Boutons

Open selected
Construit un JSON avec l’état de chaque vanne : true pour les cases cochées (ouvert), false pour les autres, puis publie sur hmi/actuators_request.
Close all
Envoie la commande qui ferme toutes les vannes (indépendamment des cases).

Le détail exact du JSON est défini dans le module applicatif ; en cas de doute, corrélez avec le moniteur MQTT.
