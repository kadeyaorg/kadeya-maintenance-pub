Robot View tab

Read-only SCARA visualization for Dispense and Return arms.

Home Dashboard PLC Reset Doors Robots Move Sequencer Robot View Valves Dispense Dispense Shelves Washer Syrup Software HMI MQTT Monitor

Data source

Subscribes to robot_dispense/state, robot_dispense/status, robot_return/state, and robot_return/status. No commands are sent from this tab.

Display

Each panel shows arm geometry from joint angles, gripper opening, Z indicator, and a dashed target when move_flag is set. Status badge: Disconnected / Idle / Ready / Moving.
