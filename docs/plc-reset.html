PLC Reset tab

Publishes to MQTT topic hmi/plcreset (PLC reset command).

Actions on this tab change machine state. Use them only if you know what you are doing or under Kadeya support instruction.

Maintenance mode

The button label follows plc/state (IsMaintenanceMode). Clicks publish {"MaintenanceRequired":true} or false on hmi/plcreset.

PLC maintenance mode / Maintenance allowed
From plc/state: IsMaintenanceMode drives Enter vs Exit; maintenance_allowed is shown for context.
Maintenance required (hmi/plcreset)
Read-only echo of the last MaintenanceRequired on hmi/plcreset.
Enter maintenance mode
When maintenance_allowed is true, one confirmation. When false (or plc/state not received yet), two confirmations before publish.
Exit maintenance mode
When IsMaintenanceMode is true. One confirmation, then {"MaintenanceRequired":false}.

Checkboxes (Reset Flags)

They do nothing by themselves: they define the payload that will be sent with the SEND RESET button.

Global Machine
Includes the global machine flag in the reset payload.
Return Machine
Sets returnmachine in the command.
Dispense Machine
Sets dispensemachine in the command.
Declare Empty Machine
Declares the machine empty (per PLC protocol).
Torque Max Motor
Checked by default in the UI; includes the max motor torque flag.
Safety Ack
Safety-related acknowledgment.
Electrical Measures
Electrical measures (corresponding flag).
Safety Restart
Safety restart.
PLC Restart
PLC restart.
PC Reboot
Reboots the onboard PC (major impact on the machine).
PC Stop
Stops the onboard PC (major impact).

Send button

SEND RESET
Opens a confirmation dialog. If you confirm, builds JSON from the checked boxes, publishes to hmi/plcreset, and shows the result (success or publish error) under the button and in the status bar.
