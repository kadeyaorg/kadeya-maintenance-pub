Onglet PLC Reset

Publication sur le sujet MQTT hmi/plcreset (commande de reset PLC).

Accueil Dashboard PLC Reset Doors Robots Valves Dispense Washer Syrup Software MQTT Monitor

Les actions de cet onglet modifient l’état de la machine. Ne les utilisez que si vous savez ce que vous faites ou sur consigne du support Kadeya.

Cases à cocher (Reset Flags)

Elles ne font rien seules : elles définissent le contenu de la commande qui sera envoyée avec le bouton SEND RESET.

Global Machine
Inclut le drapeau « machine globale » dans la charge utile de reset.
Return Module
Cible le module retour dans la commande.
Dispense Module
Cible le module distribution.
Declare Empty Machine
Déclare la machine vide (selon protocole PLC).
Torque Max Motor
Cochée par défaut dans l’UI ; inclut le drapeau couple moteur max.
Safety Ack
Accusé / acquittement lié à la sécurité.
Electrical Measures
Mesures électriques (drapeau correspondant).
Safety Restart
Redémarrage sécurité.
PLC Restart
Redémarrage PLC.
PC Reboot
Redémarrage du PC embarqué (impact majeur sur la machine).
PC Stop
Arrêt du PC embarqué (impact majeur).

Bouton d’envoi

SEND RESET
Ouvre une boîte de confirmation. Si vous validez, construit le JSON à partir des cases cochées, publie sur hmi/plcreset, et affiche le résultat (succès ou erreur de publication) sous le bouton et dans la barre d’état.
