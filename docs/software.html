Onglet Software

Mises à jour logicielles, redémarrage global et supervision des processus (agent logiciel).

Accueil Dashboard PLC Reset Doors Robots Valves Dispense Washer Syrup Software MQTT Monitor

L’interface rappelle que les mises à jour manuelles et le redémarrage affectent tous les processus logiciels de la machine : n’utiliser que sur instruction du support Kadeya.

Updates

Run manual update
Ouvre une confirmation (mise à jour immédiate, possibles redémarrages de services). Si vous validez, publie {"update":true} sur software_agent/update ; le résultat s’affiche sous le bouton.

Restart

Restart all software
Confirmation obligatoire (délai possible 30–60 s). Publie {"restart":true} sur software_agent/restart_all.

Zones sans bouton d’action

Software agent
Libellés d’état de l’agent (état, raison, run id, horodatage, détails) alimentés par les messages software_agent/status (et champs associés).
Processes (tableau)
Liste les instances à partir de software_agent/software_status : nom manifeste, statut, PID, identifiant logiciel, ligne de commande. La coloration de la colonne Status met en évidence running / stopped / autre.
