Software tab

Software updates, global restart, and process supervision (software agent).

Home Dashboard PLC Reset Doors Robots Valves Dispense Washer Syrup Software MQTT Monitor

The UI warns that manual updates and restart affect all machine software processes: use only under Kadeya support instruction.

Updates

Run manual update
Opens a confirmation (immediate update, possible service restarts). If you confirm, publishes {"update":true} to software_agent/update; result text appears under the button.

Restart

Restart all software
Confirmation required (allow 30–60 s). Publishes {"restart":true} to software_agent/restart_all.

Non-action areas

Software agent
Agent status labels (state, reason, run id, timestamp, details) fed by software_agent/status messages (and related fields).
Processes (table)
Lists instances from software_agent/software_status: manifest name, status, PID, software id, command line. Status column coloring highlights running / stopped / other.
