Robots tab

Control of Return and Dispense arms (MQTT prefixes robot_return and robot_dispense).

Home Dashboard PLC Reset Doors Robots Valves Dispense Washer Syrup Software MQTT Monitor

Buttons marked ⚠ require confirmation before send. “Home ▾” opens a dialog to pick the joint.

Common actions (each robot: Return / Dispense)

Init
Sends the arm controller init command (no extra confirmation).
Unlock
Unlocks the arm.
Pause
Pauses motion.
Resume
Resumes after a pause.
Home ▾
Opens “Select Joint”: Joint 1 through Joint 4. OK sends the home command for the selected joint; Cancel aborts.
Stop ⚠
Stop; confirmation required, then publish.
Reset Error ⚠
Error reset; confirmation required.
Brake ON ⚠
Brakes engaged; confirmation required.
Brake OFF ⚠
Brakes released; confirmation required (mechanical risk).

Sequencer

Reset sequencer (Return Robot)
Publishes the sequencer error reset command to robot_return_sequencer/reset_error.
Reset sequencer (Dispense Robot)
Publishes to robot_dispense_sequencer/reset_error.
