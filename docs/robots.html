Onglet Robots

Contrôle des bras Return et Dispense (préfixes MQTT robot_return et robot_dispense).

Accueil Dashboard PLC Reset Doors Robots Valves Dispense Washer Syrup Software MQTT Monitor

Les boutons marqués ⚠ demandent une confirmation avant envoi. « Home ▾ » ouvre une boîte de dialogue pour choisir l’articulation.

Actions communes (chaque robot : Return / Dispense)

Init
Envoie la commande d’initialisation du contrôleur du bras (sans boîte de confirmation supplémentaire).
Unlock
Déverrouille le bras.
Pause
Met le mouvement en pause.
Resume
Reprend après une pause.
Home ▾
Ouvre la fenêtre « Select Joint » : liste Joint 1 à Joint 4. OK envoie la commande de home pour l’articulation choisie ; Cancel annule.
Stop ⚠
Arrêt ; confirmation requise puis publication.
Reset Error ⚠
Réinitialisation d’erreur ; confirmation requise.
Brake ON ⚠
Freins activés ; confirmation requise.
Brake OFF ⚠
Freins relâchés ; confirmation requise (risque mécanique).

Séquenceur

Reset sequencer (Return Robot)
Publie la commande de reset d’erreur sur robot_return_sequencer/reset_error.
Reset sequencer (Dispense Robot)
Publie sur robot_dispense_sequencer/reset_error.
