Control of Return and Dispense arms (MQTT prefixes robot_return and robot_dispense).

Buttons marked ⚠ require confirmation before send. “Home ▾” opens a dialog to pick the joint.

Jog

Select the active robot (Dispense or Return), mode (Cartesian or Articular), parameters D, lr, roughtly, then use − / + / Go per axis. Jog publishes {robot}/move with gripper sentinel values when not jogging grip axes.

Live status and pose

Each robot group shows MQTT status and state fields updated while connected.

Standard commands (each robot)
