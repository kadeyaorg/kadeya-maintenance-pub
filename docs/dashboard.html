Rôle de l’onglet

Le Dashboard agrège des informations reçues par MQTT : cartes d’état pour le PLC, le module Dispense, le retour (Return), le lave-bouteilles (Washer), des compteurs / indicateurs machine, et une zone « Active issues » qui résume les blocages ou erreurs détectés.

Cet onglet ne contient aucun bouton d’action : tout est affiché en continu (rafraîchissement périodique lorsque la session est connectée).

Zones affichées

Cartes PLC, Dispense, Return, Washer
Point coloré et libellé d’état ; la couleur reflète des mots-clés dans le texte (OK / avertissement / erreur).
Métriques (Dirty Bottles, Clean Bottles, Washer Index, Washer Opened, Maintenance Mode, Blocking Reason)
Valeurs issues de l’état machine ; « Blocking Reason » explique souvent pourquoi la maintenance ou un mouvement est bloqué.
Active issues
Liste synthétique des messages d’erreur ou de blocage (PLC, Return, Dispense) lorsqu’ils sont présents ; sinon message indiquant qu’il n’y a pas de problème actif.
