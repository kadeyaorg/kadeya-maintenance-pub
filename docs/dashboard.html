Tab purpose

The Dashboard aggregates information received over MQTT: status cards for the PLC, Dispense module, Return, bottle washer (Washer), machine counters / indicators, and an “Active issues” area summarizing detected blocks or errors.

This tab has no action buttons: everything updates continuously (periodic refresh while the session is connected).

Displayed areas

PLC, Dispense, Return, Washer cards
Colored dot and status label; color reflects keywords in the text (OK / warning / error).
Metrics (Dirty Bottles, Clean Bottles, Washer Index, Washer Opened, Maintenance Mode, Blocking Reason)
Values from machine state; “Blocking Reason” often explains why maintenance or motion is blocked.
Active issues
Short list of error or block messages (PLC, Return, Dispense) when present; otherwise a message that there is no active issue.
