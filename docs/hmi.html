Maintenance

Enter machineId and use Open maintenance page so the on-machine HMI can open maintenance (same path as a validated operator scan).

Unstick HMI flow

If the HMI stays on dispense-in-progress or return-in-progress, these buttons send two state messages 1.5 seconds apart:

Clear stuck Dispense
dispense/state: first COMPLETED, then READY (with bottleId 00001, errorCode 0).
Clear stuck Return
return/state: first ACCEPTED, then READY (same fields).
