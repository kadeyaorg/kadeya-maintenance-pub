MQTT Monitor tab

Near real-time log of MQTT messages the UI is subscribed to (view / filter).

Filter fields

Topic
Filters visible rows: only messages whose topic contains the entered text (case-insensitive) remain. Leave empty to show everything.
Search
Filters on a free string (timestamp, topic, or payload), case-insensitive.

Controls

Pause (checkbox)
When checked, the display stops updating with new messages (the buffer still receives in the background up to an internal limit; if saturated, some messages may be dropped).
Clear
Clears the buffer and on-screen log; resets the message counter to zero.

Text area

Each line is one message: timestamp, topic (special formatting in the Windows UI), and payload JSON compacted to one line when valid JSON.

The counter on the right shows how many messages are kept in the buffer (capped; oldest entries are removed when the limit is exceeded).
