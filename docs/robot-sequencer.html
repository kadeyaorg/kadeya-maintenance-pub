Sequence storage

Save to file
Exports the named sequence to a JSON file (native “Save as” dialog) and keeps it in the in-app list. The file uses the legacy robot_sequences.json format ({ "name": [moves…] }).
Load file…
Opens a native file picker and imports sequences from a JSON file (compatible with the legacy robot_sequences.json). Imported sequences are added to the in-app list.
Saved sequences (dropdown)
Selecting a saved sequence loads it instantly into the editor.
Delete
Removes the named sequence from the in-app list.

Playback

NEXT
Sends the current move to robot_dispense/move and advances the index.
PLAY
Runs the list automatically; waits until moving is false (2 min timeout per step).
Stop robot
Publishes robot_dispense/stop and stops auto-play.

Copy position

Fills the move form from the latest robot_dispense/state (articular mapping: Z, A1, A2, R, gripper).
