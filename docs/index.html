À propos de l’application

Go Maintenance est une console Windows qui se connecte au broker MQTT de la machine pour afficher l’état (dashboard), envoyer des commandes de maintenance (portes, robots, vannes, etc.) et inspecter les messages MQTT reçus par l’UI.

Écran de connexion (avant les onglets)

Check for updates
Ouvre la page des releases GitHub (dépôt public de distribution) dans le navigateur par défaut.
Boutons « IP : port » (connexions récentes)
Remplissent automatiquement les champs IP et port pour une reconnexion rapide (jusqu’à 3 entrées).
Connect to Machine
Tente la connexion au broker, abonne l’application aux sujets MQTT nécessaires, enregistre la connexion dans les récents, puis affiche la barre latérale et les onglets.

En-tête après connexion

Check for updates
Même comportement que sur l’écran de connexion.
Change Machine
Revient à l’écran de connexion (déconnexion MQTT, réinitialisation de l’affichage).
Disconnect
Demande une confirmation puis déconnecte et revient à l’écran de connexion.

Navigation latérale (onglets)

Chaque entrée ouvre l’onglet correspondant. La page détaille les contrôles propres à chaque onglet :

  • Dashboard — vue d’ensemble et alertes
  • PLC Reset — drapeaux de reset PLC
  • Doors — ouverture / fermeture des portes
  • Robots — bras Return et Dispense
  • Valves — solénoïdes
  • Dispense — recettes de test
  • Washer — lave-bouteilles
  • Syrup — niveaux BIB
  • Software — mises à jour et redémarrages
  • MQTT Monitor — journal des messages
