Syrup tab

BIB level management (F1–F4) — commands on hmi/syrup/command.

F1–F4 rows

Each row shows a progress bar (remaining percent), “remaining / capacity” text in mL, and a visual alert when the level is very low.

Adjust
Opens “Adjust Syrup Level”: enter remaining (mL) and capacity (mL). OK validates (remaining between 0 and capacity; capacity between 1000 and 20000 mL) and sends the adjust command for the row (F1–F4). Cancel closes without sending.
Replace BIB
Asks for confirmation, then sends a command that resets the line BIB to full capacity (capacity read from current state, or default 9500 mL if received capacity is 0).

Percentages and mL reflect MQTT data; after a command, the display updates when the machine publishes new state.
