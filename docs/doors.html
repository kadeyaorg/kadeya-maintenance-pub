Return Door

Open door
Publie {"open":true} sur hmi/return/door pour demander l’ouverture.
Close door
Publie {"open":false} sur hmi/return/door pour demander la fermeture.

Dispense Door

Open door
Publie {"open":true} sur hmi/dispense/door.
Close door
Publie {"open":false} sur hmi/dispense/door.

Respectez les consignes de sécurité sur site : ne commandez les portes qu’avec une zone dégagée et une procédure validée.
