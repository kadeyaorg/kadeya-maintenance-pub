Door open / close commands over MQTT.

Return Door

Open door
Publishes {"open":true} to hmi/return/door to request opening.
Close door
Publishes {"open":false} to hmi/return/door to request closing.

Dispense Door

Open door
Publishes {"open":true} to hmi/dispense/door.
Close door
Publishes {"open":false} to hmi/dispense/door.

Follow on-site safety rules: command doors only with a clear area and an approved procedure.
