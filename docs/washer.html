Washer tab

Bottle washer: start command and MQTT state display.

Button

Washer Relay
Publishes to dishwasher/start JSON such as {"program":1} to start the selected program. Result text (success / error) appears under the button and in the status bar.

Information fields (not buttons)

Status
Bottle washer state, from dishwasher/status (0 = idle, 1 = washing, 2 = error) and the PLC plc/washer state.
Level
Detergent levels from dishwasher/level (rinse aid, disinfect, cleaner — in %).
PLC index / Hood up / Hood down / PLC state
Live values from plc/washer (index, capacity, hood position, current process).

Hobart Cloud (desktop only)

The washer is a Hobart dishwasher. When configured, this panel pulls cloud telemetry from the Hobart Cloud-to-Cloud API (separate from the machine MQTT broker, refreshed on a slow cadence imposed by Hobart: temperatures every 60 s, errors every 30 s, daily data on demand).

Settings
Enter the Hobart account credentials (username / password) and the Hobart serial of this machine’s washer. The serial is bound to the connected machine (one washer per machine, distinct serials), so each machine keeps its own binding. Credentials are stored locally on the device, never in the app bundle.
Discover machines
Lists the machines registered to the Hobart account so you can pick the correct serial.
Connection status
A green / red dot shows whether the Hobart cloud connection (ping) succeeds.

Once configured, the panel shows:

  • Temperatures — wash and rinse temperature, plus the last program run.
  • Active errors — current Hobart errors (critical ones highlighted in red).
  • Daily consumption — water, energy, detergent and rinse-aid usage for the day.
  • Hygiene protocol — recent program runs with their timestamps.

Cloud data complements (does not replace) the live MQTT washer values above. The Hobart panel is available on the desktop app only.
