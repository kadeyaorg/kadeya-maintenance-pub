Bouton

Washer Relay
Publie sur dishwasher/start un JSON du type {"program":1} pour lancer le programme indiqué. Le libellé de retour (succès / erreur) s’affiche sous le bouton et dans la barre d’état.

Champs d’information (pas des boutons)

Status
État du lave-bouteilles (sujet d’état agrégé côté application).
Level
Niveau remonté par MQTT (dishwasher/level dans la pile de données).
Error
Code ou message d’erreur courant (dishwasher/error).
